Feature: This is guideline for the feature file keywords

Scenario: Scenario is the high level description of the test
Given is used to mention the preconditions, avoid having user interaction
When is used to describe the user actions
Then is used to observe the expected output/ validation, which should be related business value or benefit
	And This is used as an addition to the previous steps and represents positive statements
	But This is used as an addition to the previous steps and represents negative statements
